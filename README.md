# goit-algo-hw-02

Завдання 1.

Програма, яка імітує приймання й обробку заявок: вона автоматично генерує нові заявки 
(ідентифіковані унікальним номером або іншими даними), додає їх до черги, а потім 
послідовно видаляє з черги для "обробки", імітуючи таким чином роботу сервісного центру.

Завдання 2.

Необхідно розробити функцію, яка приймає рядок як вхідний параметр, додає всі його символи 
до двосторонньої черги (deque з модуля collections в Python), а потім порівнює символи 
з обох кінців черги, щоб визначити, чи є рядок паліндромом. 
Програма повинна правильно враховувати як рядки з парною, так і з непарною кількістю символів, 
а також бути нечутливою до регістру та пробілів.

Ось приклади запуску другого додатку з довгими паліндромами:

python task2.py A man, a plan, a canal: Panama

python task2.py Do geese see God?

python task2.py Lisa Bonet ate no basil
